Yu-Gi-Oh!
==================

Descrição
---------

A partir da leitura de cartões RFID no Arduino, usaremos o Unity3D para exibir os modelos 3D das imagens das cartas.

Componentes
-----------
Radiofrequência, RFID, Arduino UNO, Unity3D.
